## 项目定位

RedConvert 是一个基于 Electron + React + TypeScript 的 AI 原生桌面应用，核心功能包括：
- **智囊团（Advisors）** - 多 Agent 协作系统
- **创意聊天室（群聊）** - 总监引导的多角色讨论
- **稿件编辑器** - CodeMirror 6 驱动的专业写作体验
- **知识库管理** - 小红书/YouTube/文本片段管理
- **AI Agent 驱动** - Agentic Search 与角色扮演
