# 商媒运营助手 Cohmira 品牌方案

## 名称

- 中文名：**商媒运营助手**
- 英文名：**Cohmira**
- 推荐读法：`ko-MEE-ra`，中文近似“科米拉”
- 英文构词：`cohesion`（协同）+ `mira`（视野与未来感）
- 品牌短句：**让电商与自媒体运营交给智能助手协同完成。**

“商媒”直接对应电商与自媒体业务；“运营”覆盖素材采集、选题、内容生产、账号发布、复盘和经营管理；“助手”说明产品由多个 Agent 协同执行，老板与员工保留目标设定、授权和最终决策权。

## 产品命名

- 员工端：**Cohmira Workspace / 商媒运营助手 · 员工工作台**
- 老板端：**Cohmira Command / 商媒运营助手 · 老板指挥台**

## Logo 语义

- 主体轮廓是英文名首字母 `C`。
- 三个节点代表内容、渠道和交易，也代表参与协作的多个 Agent。
- 环形轨迹表达采集、创作、发布和复盘形成的运营闭环。
- 中央星芒代表老板端设定的共同目标，以及 Agent 汇聚后的增长结果。
- 圆角方形适用于 macOS 与 Windows 的应用图标体系。
